✏️ Canvas 스크롤 방지 시스템

Canvas에서만 스크롤 방지, 다른 영역에서는 자유로운 스크롤

🎯 작동 방식

Canvas 영역: 터치펜/스타일러스로 필기할 때 스크롤이 완전히 방지됩니다.

Canvas 외부: 일반적인 스크롤, 확대/축소, 모든 터치 제스처가 정상 작동합니다.

모바일 최적화: 갤럭시 S펜, 서피스 펜, 아이패드 애플펜 완벽 지원!

입력 타입: -
Pressure: 0.000
좌표: 0, 0
색상: #000000
굵기: 2px

📜 일반 스크롤 영역

이 텍스트 영역에서는 일반적인 스크롤이 완벽하게 작동합니다. 마우스 휠이나 터치 제스처로 자유롭게 스크롤할 수 있습니다.

Canvas 밖에서는 모든 브라우저 기본 동작이 정상적으로 작동합니다:

  • ✅ 세로 스크롤 (위/아래)
  • ✅ 가로 스크롤 (좌/우, 필요시)
  • ✅ 확대/축소 (Pinch zoom)
  • ✅ 텍스트 선택 및 복사
  • ✅ 우클릭 컨텍스트 메뉴
  • ✅ 드래그 앤 드롭

🎢 내부 스크롤 테스트 영역

이 박스는 자체 스크롤이 있습니다. 마음껏 스크롤해보세요!

스크롤 콘텐츠 1

여기서 스크롤이 잘 작동하는지 확인해보세요.

⚙️ 기술적 구현

CSS 방식: Canvas 컨테이너에만 touch-action: none 적용

JavaScript 방식: Canvas 영역에서만 preventDefault() 호출

이벤트 전파: Canvas 외부에서는 기본 브라우저 동작 유지

이 방식은 가장 안정적이고 직관적인 사용자 경험을 제공합니다.

📏 긴 콘텐츠 영역

페이지 스크롤 테스트를 위한 긴 영역입니다.

Canvas 밖에서는 모든 스크롤이 자유롭게 작동합니다!
